# P2HACKS令和元年 アピールシート

## プロダクト

### コンセプト
ークリスマスシーズンの料理の手助けwebサービス
### 説明
ークリスマスシーズンになると、家族であったり友達であったり、みんなの前で料理を披露する機会が増えてきます。そんな時に人気のレシピがすぐわかるようなサービスがあれば便利だと考え、このWebサービスを考えました

### 推しポイント
ークリスマス以外にも利用できる仕様となっています。
